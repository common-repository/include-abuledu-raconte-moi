Ce plugin permet d'intégrer directement un flux provenant de http://raconte-moi.abuledu.org/ au même titre qu'une vidéo youtube ou un autre site déjà "compatible wordpress".

== Description ==

Après avoir installé ce plugin, il vous suffit d'ajouter sur une adresse d'une page du site raconte-moi.abuledu.org sur ligne vide d'un article ou d'une page de votre wordpress pour qu'elle soit intégrée.

Par exemple si j'ajoute une ligne avec

http://raconte-moi.abuledu.org/w/7681-chloe_a_la_plage-3

cette ligne sera remplacée par une iframe contenant l'histoire de chloé à la plage.

Pour plus de détails vous pouvez aller sur le site des dévelopeurs : https://redmine.ryxeo.com/projects/leterrier-racont/wiki/Plugin_pour_wordpress

== Installation ==

* Installez ce plugin
* Activez-le
* Et maintenant dans une page ou un article, ajoutez une ligne avec uniquement une adresse http://raconte-moi.abuledu.org/.... (rien d'autre sur la ligne) et c'est tout !
